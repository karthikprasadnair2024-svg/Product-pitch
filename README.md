Walkthrough
Click the Prompt Builder button beside the chat box.
Enter a goal.
Select the AI's Role, Tone, and Format.
Add context (if required). If files are attached in chat, the builder automatically detects them.
Click Generate Prompt.
Review or edit the generated prompt in Prompt Preview.
Optionally click Enhance to further refine the prompt.
Submit the final prompt to receive the AI response.

Prototype Notes
This prototype is intended to demonstrate the user experience, workflow, and interactions of the Smart Prompt Builder.
The Generate Prompt and Enhance actions are not yet integrated with the OpenAI API.
To demonstrate the workflow, these actions currently return template-based placeholder prompts rather than AI-generated prompts.
In the proposed implementation, both actions will invoke the OpenAI API to generate and refine prompts dynamically based on the user's structured inputs and any detected chat attachments.
The primary objective of this prototype is to validate the UI, interaction flow, and overall user experience, rather than the backend AI implementation.
